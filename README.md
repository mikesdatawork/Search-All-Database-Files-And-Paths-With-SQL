## About-Process

<p>Search through all the Data files to find a specific data file path.
Here's a quick script to search through ALL the database files, and paths to find a specific file.</p>      


## SQL-Logic
```SQL
use master;
select
'database' = db_name(database_id)
, [file_id]
, 'name' = name
, 'path' = physical_name
from
sys.master_files
where
db_name(database_id) not in ('master', 'msdb', 'model', 'tempdb') 
and physical_name like '%MyWildCard%'
order by
db_name(database_id), [file_id], name, physical_name

```
